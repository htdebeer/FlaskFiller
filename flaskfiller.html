FlaskFiller

FlaskFiller is made as part of a research on teaching instantaneous speed in 5th grade. During that research, four teaching experiments have been performed, each time with an improved version of the FlaskFiller software. The version you are working with now is the latest incarnation.

Software

FlaskFiller is free software and is released under the GPL version 3 license. This means you can use, adapt, and distribute FlaskFiller as long as these adaptations are also published under the same license.

FlaskFiller is made using the following libraries and programs:

  • Raphaël.js for the simulation component
  • d3 for the graph component
  • Bootstrap and jQuery for the menu, tabs, and settings page
  • Font awesome for the icons
  • Browserify to create the FlaskFiller javascript library
  • and pandoc to generate the standalone verion

You can download FlaskFiller from heerdebeer.org/Software/flaskfiller/.
